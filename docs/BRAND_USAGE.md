# Brand and Identity Usage

The names `BSFF`, `neuron7xLab`, the `âŠ›` marker, and author identity markers are provenance and attribution identifiers.

Allowed:

- Mentioning BSFF as upstream software.
- Forking with clear attribution.
- Publishing derivative work that states it is modified from BSFF.

Not allowed:

- Implying official endorsement by Yaroslav Vasylenko / neuron7xLab.
- Rebranding a derivative as the original BSFF project.
- Removing copyright, NOTICE, citation, or SPDX metadata.
- Using the author identity or project markers to confuse users about origin.

Recommended fork naming:

```text
<your-project-name>, derived from BSFF by Yaroslav Vasylenko / neuron7xLab.
```
